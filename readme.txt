The System requirements will be at least:Windows 7 Service Pack 1, Windows Server 2008 R2, Windows Server 2008 R2 SP1, Windows 8, Windows Server 2012

The supporting architecture will be:
1. x86
2. x64

The hardware requirements will be at least:
1. Pentium 1 GHz 1 core
2. 1 GB RAM
3. 40 GB Hard Disk
 
The prerequisites are followed:
1. Microsoft .NET Framework 4.0
2. Microsoft Visual Studio C++ 2008 redistibutables
3. Microsoft Visual Studio C++ 2010 redistibutables
4. Windows OpenSSL 
5. Windows Installer 3.1

The installation will include the above 1-4 prerequisites if the target system is not installed yet.If the target system has installed previous version of gConnect, the previous version will be uninstalled and reinstalled the latest version.
